Some of my Recent Projects

CALCULATOR

Project Title: Simple Calculator using Python and Tkinter

Description: A basic calculator application built with Python's Tkinter library, allowing users to perform arithmetic operations like addition, subtraction, multiplication, and division.

Features: User-friendly GUI with buttons for numeric input and operations, display area for showing results, and clear button to reset the calculator.

TEMPERATURE CONVERTER

Project Title: Temperature Converter in C

Description: A simple console-based application written in C that converts temperatures between Celsius, Fahrenheit, and Kelvin.

Features: User-friendly interface with options to input temperature in one unit and get the equivalent temperature in the other two units, ensuring accurate conversions based on standard formulas.

GPA CACULATOR

Project Title: GPA Calculator using HTML and JavaScript

Description: A simple web-based GPA calculator where students can input their grades and credit hours to calculate their GPA.

Features: User-friendly interface with input fields for grades and credit hours, JavaScript functions to compute GPA based on provided inputs, and a display area to show the calculated GPA instantly.
